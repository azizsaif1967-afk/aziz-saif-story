Technology Stack

Built with the essentials — nothing more

A deliberately lean stack. No build step, no framework lock-in. Edit a single HTML file, push to GitHub, the site is live.

Frontend
HTML5 + CSS3
Hand-crafted semantic HTML with custom CSS variables for design tokens. No CSS framework.
Interactions
Vanilla JavaScript
Lightweight ES6+ for the floating catalogue, gallery lightbox & smooth scroll. Zero dependencies.
Typography
Google Fonts
Playfair Display for headings & Inter for body — loaded with preconnect for performance.
Hosting
GitHub Pages
Free, fast, global CDN. Deployment is a git push. CNAME mapped to globalpioneer.xyz.
Images
CloudFront CDN
Brochure pages served via AWS CloudFront with optimized PNG compression for instant load.
SEO
Schema.org + OG
Open Graph tags, structured data & semantic HTML for organic ranking on Google & LinkedIn.
Build Process

From brief to live URL in 72 hours

A repeatable process — brief, content extraction, design system, build, deploy, iterate.

Brief & competitor scan

30-minute call to capture the company story, target customer, key services & competitive landscape. Reviewed 5 GCC trading company sites for visual reference.

Content & brochure extraction

Extracted the existing 24-page brochure into individual high-res images, organized by service area (sourcing, showrooms, design, franchise).

Design tokens & colour system

Established a navy + gold palette with CSS variables — primary #0a1628, accent #c9a84c. Defined typography scale & spacing rhythm before writing any HTML.

Floating catalogue interaction

Custom JS + CSS to build a page-flip catalogue experience that mimics the printed brochure — without any heavy library like Turn.js.

Mobile-first responsive pass

Started layouts at 320px, scaled up to 1440px. Tested on actual devices in the office before pushing to production.

Deploy & analytics

GitHub Pages deploy, CNAME setup at Namecheap for globalpioneer.xyz, Google Analytics 4 wired in for traffic insights.
