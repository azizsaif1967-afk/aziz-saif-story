Stack Behind the Scenes

What it’s built on

Modern, lean, AI-native. Every layer chosen for speed of iteration over enterprise complexity.

Frontend
Next.js 14 + Tailwind
App Router, React Server Components for fast, SEO-friendly listing pages.
Database
Supabase (Postgres)
Postgres + pgvector for semantic search, Row Level Security for agent & admin separation.
AI Engine
Claude + GPT-4 Vision
Claude for copy, classification & lead qualification. GPT-4 Vision for floor plan analysis.
Automation
Manus + Cron
Manus orchestrates multi-step automations (scrape → enrich → publish). Cron triggers daily digests.
Messaging
Twilio WhatsApp + Resend
WhatsApp for buyer & agent comms, Resend for transactional & marketing email.
Hosting
Vercel + CDN
Edge functions for instant page loads across UAE, MENA & expat source markets.

Lessons From The Build

What this project taught us

Three principles that shape how we build AI-first products for the GCC.

Lesson 01
Automate the boring 80%
Listing copy, classification, photo tagging — all things an agent shouldn’t spend time on. AI handles them in seconds, agents stay focused on closing deals.
Lesson 02
WhatsApp is the UI
In the UAE, buyers & agents transact on WhatsApp. Building a pretty webapp without WhatsApp-first comms is building for the wrong reality.
Lesson 03
Manus runs the loops
Multi-step automations (scrape → enrich → publish → notify) are exactly what Manus is built for. We orchestrate, AI does the work.
