Raw HTML (if you need to paste into a signature editor)

Gmail

  1. Click the Copy signature button above.
  2. Open Gmail → Settings → See all settings → General.
  3. Scroll to Signature → Create new → paste.
  4. Set as default for New mail & Reply/Forward.
  5. Save changes.

Outlook (desktop)

  1. File → Options → Mail → Signatures.
  2. New → paste the copied signature.
  3. Assign to New & Replies accounts.
  4. OK to save.

Apple Mail

  1. Mail → Settings → Signatures.
  2. Create a new signature, paste, and uncheck Always match my default font.
  3. Drag the signature to the account you want.
